听译 Tingyi 桌面版

【启动】双击 Start Tingyi.bat 或 Tingyi.exe

【使用】
1. 点进微信 / Cursor 输入框
2. 按 F9（默认热键，不与输入法切换冲突）
3. 说话，停顿约 1.2 秒自动结束并粘贴

【改热键】同目录 .env 文件：
  TINGYI_HOTKEY_RECORD=f9
  可选：ctrl+alt+v 、 f8 、 ctrl+alt+.

【改停顿时间】觉得结束太快：
  TINGYI_VAD_MIN_SILENCE_MS=500
  （单位毫秒，500=0.5秒；若句尾被截断可改为 800~1200）

【注意】勿删 models/；热键无效可管理员运行
